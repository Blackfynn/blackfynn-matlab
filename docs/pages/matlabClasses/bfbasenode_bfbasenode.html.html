  BFBaseNode Constructor for the BFBASNODE CLASS
    OBJ = BFBaseNode(SESSION, 'id') creates an object of class
    BFBaseNode where SESSION is an object of class BFSESSION,
    and 'id' is the Blackfynn ID of the object.
