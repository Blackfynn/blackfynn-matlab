  listLayers returns a MATLAB Table with annotation layer info
    LAYERS = listLayers(OBJ) returns a MATLAB table
    representing the annotation layers that exist for the
    timeseries object.
 
    For example:
        ts.listLayers()
 
          +----------+------------+-------------------+
          | ID       | Name       | Description       |
          +----------+------------+-------------------+
          | layer_id | layer_name | layer_description |
          +----------+------------+-------------------+
