  delete_layer Removes layer associated with timeseries object
 
  Args:
        id (int): ID of the layer to be deleted
 
  Examples:
 
            delete layer with ID 384 and associated to ``ts``,
            a timeseries object::
 
                >> ts.delete_layer(384)
