  BFDataset Constructor of the BFDataset Class
    OBJ = BFDataset(SESSION, 'id', 'name') creates a
    BFDataset instance where SESSION is the BFSESSION object,
    'id' is the id of the dataset and 'name' is the name of the
    dataset.
