The Blackfynn Toolbox

The Blackfynn toolbox provides Blackfynn users with advanced tools to interact with their scientific data and meta-data on the Blackfynn platform. The toolbox provides methods to navigate datasets on the platform, to generate/modify and access the Blackfynn Knowledge Graph and to interact with a number of data modalities directly over a streaming interface. This allows random access into large data that are stored on the platform.

Developer documentation for the MATLAB client can be found online: MATLAB Developer Documentation. Here, you will find instructions on how to install and configure the MATLAB toolbox. The developer site also provides documentation on all Blackfynn MATLAB classes and methods and contains a number of tutorials to get you started with the toolbox.
